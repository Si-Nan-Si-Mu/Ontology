# 商业 IP、品牌与商标指引（POG）

本文档用于**产品化与对外传播**时的命名、视觉与知识产权规划；**不构成法律意见**。商标注册、异议与侵权判断须委托**具备资质的商标代理机构或律师**，并在目标法域检索官方数据库。

---

## 1. 品牌内核（建议固定对外表述）

| 要素 | 建议内容 |
|------|-----------|
| **英文主名** | **POG**（缩写） / 全称 **Personality Ontology Graph** |
| **中文全称** | **人格本体图**（与学术/技术语境一致，利于答辩与 B 端沟通） |
| **中文商业副名（可选）** | **格谱**（两字、易记、暗示「人格 + 图谱」；注册前须做**近似商标检索**） |
| **Slogan（候选）** | 「人格可结构化，证据可查询。」 / 「Structured persona. Auditable evidence.」 |
| **品类一句话** | 面向心理科技与企业场景的**人格—社会情境图数据与推理层**（非医疗诊断工具）。 |

**避免**：在宣传中暗示「医疗诊断」「官方人格鉴定」「替代量表」等未经验证的合规风险表述；与 `docs/ETHICS_AND_DATA.md` 保持一致。

## 2. 商标策略（概要）

### 2.1 可考虑的标识类型

1. **文字商标**：`POG`、`格谱`、中英组合（如「格谱 POG」）— 按使用场景分件申请或组合申请（以代理人建议为准）。  
2. **图形商标**：仓库内提供的 **`docs/brand/pog-mark.svg`** 的图形部分（三节点三角构图 + 圆角底），可导出 PNG/PDF 提交图样。  
3. **组合商标**：图标 + 标准字横排/竖排（定稿后固定间距与最小尺寸）。

### 2.2 尼斯分类（Nice）常见指向（仅供参考）

| 类别 | 典型覆盖 | 与本项目相关性 |
|------|-----------|----------------|
| **第 9 类** | 可下载软件、计算机程序 | 后端/客户端软件产品 |
| **第 42 类** | 软件即服务（SaaS）、平台、技术咨询、数据存储与检索服务 | **主类建议重点评估** |
| **第 35 类** | 广告、商业管理、在线市场（若涉及平台撮合） | 有 ToC/平台化时考虑 |
| **第 41 类** | 教育、培训、出版（若做课程/工作坊品牌） | 培训产品线时考虑 |
| **第 44 类** | 部分法域含心理咨询等；**措辞与资质要求极严** | 仅在与持证主体合作且合规审查通过后再评估 |

具体商品/服务项表述须由代理人按《类似商品和服务区分表》撰写，以提高核准率。

### 2.3 中国大陆申请流程（极简）

近似检索（商标局数据库 / 第三方工具）→ 准备申请人主体资格与图样 → **递交申请** → 形式审查 → 实质审查 → 初步审定公告（**3 个月异议期**）→ 核准注册。周期常为数月至一年以上，依审查负荷而定。

### 2.4 海外与防御性布局

- **马德里体系**：在已有国内基础申请/注册的前提下，通过 WIPO 指定多国（策略与成本由代理人规划）。  
- **美国 USPTO / 欧盟 EUIPO**：若目标市场明确，可单独规划。  
- **防御性注册**：核心类别 + 关联类别；注意**三年/五年不使用**被撤销风险，需保留使用证据。

## 3. 视觉识别规范（VI 摘要）

### 3.1 图形标（`docs/brand/pog-mark.svg`）

- **含义**：三节点与连线抽象表示「主体—关系—假设」图结构；深色底增强在浅色网页顶栏与幻灯片中的对比度。  
- **安全区**：图形外缘预留 **≥ 8% 边长** 的留白，不得贴边裁切。  
- **最小尺寸**：数字界面建议 **≥ 24px** 边长；印刷建议 **≥ 8mm**。  
- **单色稿**：对外印刷黑白时，可将渐变改为 **实色 #4338ca** 线条 + **#ffffff** 节点，或反白版本（深蓝底 + 白线白点）。

### 3.2 标准字（建议）

- **英文**：*Inter*、*DM Sans* 或 *Source Sans 3*（SIL OFL，商用友好）。  
- **中文**：*思源黑体 Source Han Sans* 或授权后的品牌定制字。  
- **字重**：标题 SemiBold / Bold；正文 Regular。

### 3.3 主色与辅助色（与现有前端气质对齐）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#4338ca` ~ `#4f46e5` | 靛蓝，理性、可信 |
| 辅色 | `#7c3aed` | 紫，创新与 AI 联想 |
| 背景深 | `#0f172a` | 与图形标底一致 |
| 警示/合规 | `#b91c1c` | 仅用于免责声明、错误态 |

## 4. 其他商业 IP（与商标并行）

| 类型 | 说明 |
|------|------|
| **软件著作权** | 对核心代码与文档做登记（中国版权保护中心等），利于融资与侵权举证。 |
| **发明专利** | 若有「图结构 + 置信度传播 + 冲突消解」等**可撰写为技术方案**的创新，可与专利代理人评估新颖性（非功能本身、而是具体技术实现）。 |
| **商业秘密** | 训练数据、客户名单、未公开的算法参数等，通过合同与访问控制保护。 |
| **域名** | 尽早注册与主名一致的 `.com` / `.cn` / `.app` 等；注意与已有商标冲突。 |

## 5. 对外使用清单（发布前自检）

- [ ] 材料中是否标注「探索性 / 非临床 / 非职业测评」等边界（见伦理文档）。  
- [ ] 是否使用**固定版本**的 Logo 与色值（避免每次活动随意改色）。  
- [ ] 是否避免与知名品牌、竞品商标**视觉或读音**高度近似。  
- [ ] 合同与网站页脚是否预留 **© 年份 权利人名称** 与商标状态说明（「™ 使用中」或「® 已注册」须与事实一致）。
